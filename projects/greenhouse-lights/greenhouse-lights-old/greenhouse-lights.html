Greenhouse Lights V1.0

I recently got sucked up into the world of gardening and have been fascinated with everything to do with it. This summer I started my own vegetable garden outdoors and had great success. Living in Canada our Winters are too harsh to grow outside so I've opted to create an indoor hydroponic setup that can be used year round. One downside to indoor gardening is it requires artificial lighting which can be tricky, but I belive I have come up with a good solution.

I bought a few 50W full spectrum LED grow lights and attached cooling as well as put them on a timer using an Arduino Nano

This is the first version of this project. If you want to see the latest version of this project Click here.


Click on the images to enlarge them.

Overview of all of the projects components. Top: 50W LED light attached to an old GPU casing for cooling. Middle: Ardunio controlled outlet with cirtuit board and Arduino Nano on top. Bottom: Outlet 120V AC power supply and 12V DC wall adapter for Arduino and GPU fan.

A closer look at the LED modual. This is the focus of the project and the component that will give my plants energy. The 50W LED produces a TON of heat, on its own with no cooling the LED would exceed 175°C. To keep it from overheating and potentially starting a fire I mounted the LED to an old GPU casing. I was able to screw the LED to the heatsink and I then connected the fan pins to a 12V line. With this cooling the LED now maintains a safer temp of around 75° C.

The circuitry for this project. Theres not too much here, it contains: A real time clock modual for keeping track of light cycles, a 18 Ω 5W resistor to slow the fan speed down and a flyback diode (not visible). The rest of the wires are for power running in and out aswell as the 5V pin to flip the relay in the outlet.

The Arduino Nano controlling everything.The Nano is powered by 12V Vin doing this saved me a power adapter because I already needed 12V to power the GPU fan. White: Vin, Black Ground, Orange: 5V, Blue: SDA and SCL (Clock pins), Yellow: D8 (Relay Pin)


A side shot of the outlet modual. There are two wires that run from the Arduino to the outlet box, Relay Pin and Ground. When the Relay Pin is given 5V from the Arduino a 5v 20a relay inside the box is closed powereing the outlet which turns on the light.

12 DC Wall adapter (Black) and 120V AC line (Grey). The two power cords for the device. I tried to keep the amount of things being pluged in to a minimum and ended up with two. There probably is a way I could have got it down to one wall plug but I'm happy with how it turned out.

This is what the light looks like when it's turned on. The light is extremely bright. It's a hard to look at without glasses. This photo was taken with ISO 50 and a cloth over the light to dim it enough to see everything on camera.

The LED moduals running temperature hovers around 75°C when the fan is running. Check out the code below, try it out for yourself and let me know what you think at colinbleslie@gmail.com

// Date and time functions using a DS1307 RTC connected via I2C and Wire lib
#include <Wire.h>
#include "RTClib.h"

#if defined(ARDUINO_ARCH_SAMD)
// for Zero, output on USB Serial console, remove line below if using programming port to program the Zero!
#define Serial SerialUSB
#endif

RTC_DS1307 rtc;

char daysOfTheWeek[7][12] = {"Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"};
int lightPin = 8;

void setup () {

#ifndef ESP8266
  while (!Serial); // for Leonardo/Micro/Zero
#endif

  Serial.begin(57600);
  if (! rtc.begin()) {
    Serial.println("Couldn't find RTC");
    while (1);
  }

  if (! rtc.isrunning()) {
    Serial.println("RTC is NOT running!");
    // following line sets the RTC to the date & time this sketch was compiled
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
  }
  pinMode(lightPin, OUTPUT);
  digitalWrite(lightPin, HIGH);
  delay(5);
} //end setup

void loop () {
  DateTime now = rtc.now();

  //--------------    DEBUG   --------------
  //    Serial.print(now.year(), DEC);
  //    Serial.print('/');
  //    Serial.print(now.month(), DEC);
  //    Serial.print('/');
  //    Serial.print(now.day(), DEC);
  //    Serial.print(" (");
  //    Serial.print(daysOfTheWeek[now.dayOfTheWeek()]);
  //    Serial.print(") ");
  //    Serial.print(now.hour(), DEC);
  //    Serial.print(':');
  //    Serial.print(now.minute(), DEC);
  //    Serial.print(':');
  //    Serial.print(now.second(), DEC);
  //    Serial.println();

  //Toggle Light
  //digitalWrite(lightPin, !digitalRead(lightPin));
  //--------------    DEBUG   --------------

  //Turns the light on at 7pm off at 7am. Light is only on during off peak hours
  if (now.hour() > 7 and now.hour() < 19) {
    digitalWrite(lightPin, LOW);
  } else {
    digitalWrite(lightPin, HIGH);
  }

  //Poll every minute
  delay(60000);
} //Loop End
